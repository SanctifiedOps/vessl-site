# vessl-site

React + TypeScript + Vite starter for the SanctifiedOps brand site.

## Prereqs
- Node 24.11.1, npm 11.2.0, and Python 3.11.5 are already available locally.

## Getting started
- Install deps: `npm install`
- Start dev server: `npm run dev` (add `-- --host` if you want LAN access)
- Lint or build: `npm run lint` / `npm run build`

## Connect to GitHub
- If you want to push to `https://github.com/SanctifiedOps/vessl-site`:  
  `git init && git add . && git commit -m "chore: bootstrap Vite React app"`  
  `git branch -M main && git remote add origin https://github.com/SanctifiedOps/vessl-site.git`  
  `git push -u origin main`
